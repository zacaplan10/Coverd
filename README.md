This is a full-stack Blackjack game where users can play against a virtual dealer using real blackjack logic. It consists of a C# ASP.NET Core Web API backend including an in-code database and a React frontend.

**Tech Stack**

**Backend:**

* ASP.NET Core Web API
* C#

**Frontend:**

* React (JavaScript)
* Basic HTML/CSS for UI
